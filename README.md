# VotingPendapat

Aplikasi voting sederhana berbasis Android + Firebase Realtime Database  
Admin dapat membuka & menutup voting, pengguna bisa memberikan suara **Setuju / Tidak Setuju** disertai alasan.

**Admin login:** `pasundan / pasundan`  
**Firebase:** gunakan file `google-services.json` milikmu di folder `app/`.
